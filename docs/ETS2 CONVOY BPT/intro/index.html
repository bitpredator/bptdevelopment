EU | BPT Simulation

"EU | BPT Simulation" è un server basato su Euro Truck Simulator 2. Supportiamo l'utilizzo delle optional mod, cio' vuol dire che puoi unirti al server senza che gli altri giocatori abbiano la mod o che sia presente nel server. Al momento puoi unirti al server tramite nome di sessione o ID

INFORMAZIONI SERVER
